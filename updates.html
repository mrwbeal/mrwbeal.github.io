Updates Feed

This page is set up as a running feed for lab milestones, conference travel, fieldwork, photos, and quick notes. New posts can be added to the top so the page reads like a simple academic timeline.

Add a new entry by copying a single .feed-item block and updating the text, date, and image.

What This Page Is For

Use this space for lighter-weight updates that do not need a full project page: conference attendance, new collaborations, paper milestones, outreach events, screenshots, travel photos, or short highlights from ongoing work.

Posting Ideas

  • Share one photo with a short caption from a conference or workshop.
  • Note when a paper is accepted, submitted, or published.
  • Highlight new datasets, talks, or outreach events.
